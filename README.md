# 🛒 Intelligent Supermarket Management System (AI)

Welcome to my complex Artificial Intelligence project. This software is a **Full-Stack** solution (FastAPI + Flet) integrating 9 AI engines to optimize supermarket operations.

## 🚀 AI Features (9 Modules)
My application uses Machine Learning algorithms for:
* **Regression & Revenue**: Sales forecasting (Random Forest).
* **Affluence**: Customer traffic prediction (Classification).
* **Storage**: Predictive analysis of stock shortages.
* **Security**: Anomaly and fraud detection at checkout.
* **Staffing**: Optimization of staff scheduling.
* **Sentiment**: Customer satisfaction analysis (NLP).
* **Clustering**: Customer segmentation (K-Means).
* **Recommendation**: Product suggestions (Cross-selling).
* **Accounting**: Automated financial statements and forecasting.

## 🛠️ Technologies Used
* **Frontend**: Flet (Python-based Flutter framework)
* **Backend**: FastAPI (Python)
* **AI/ML**: Scikit-Learn (Random Forest, K-Means), Pandas, NumPy
* **Security**: Passlib (Bcrypt) for password hashing
* **Reports**: FPDF2 for PDF invoice generation

## 📦 Installation
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the server: `uvicorn main:app --reload`
4. Run the interface: `python main_app.py`
